## Tecnologias

- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP
- Claude code
- Chatgpt





## Como executar

1. Instale o XAMPP
2. Inicie Apache e MySQL
3. Importe o banco ws_kitchen
4. Coloque os projetos na pasta htdocs
5. Acesse localhost/ws-kitchen


## Estrutura

ws-kitchen/
ws-admin/
database/
